With this module, clicking a Scene Document Link (e.g. `@UUID[Scene.cRYvlQbTdIYLQWNO]`) views the Scene, instead of opening the Configuration.

**Modifier Keys**

When holding `SHIFT` while clicking a Scene Document Link, the scene will be activated (GM only, can be disabled via a setting).
When holding `CTRL` while clicking a Scene Document Link, the Journal will be opened (provided there is one, can be disabled via a setting).

**Player View**

Players can view the scene if it appears in the scene navigation bar for them (can be disabled via a setting).

**Compendium Scenes**

A scene in a compendium can't be viewed or activated. Such a scene is ignored by this module.
